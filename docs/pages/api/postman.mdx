## Download the Authenta API Postman Collection

You can download the Postman Collection JSON and import it directly into Postman.

Or use our [Postman workspace](https://www.postman.com/authenta-ai/authenta/collection/n0wd4yd/authenta-api) to access and test the APIs online.

### 2. Import into Postman

In Postman:

- Click **Import**
- Select **File**
- Choose the downloaded collection JSON

---

### 3. Configure Authorization

Set the Authorization type to **Bearer Token** and use your Authenta API token.

Example:

```text
Authorization: Bearer YOUR_API_TOKEN
